# 油猴脚本开发模板

> 以斗鱼直播自动切换清晰度作为示例。
> forked from <https://github.com/Eished/tampermonkey-template>
> 配置了 react、tailwindcss 等。

使用此模板 [创建油猴脚本项目](https://github.com/calfzhou/tampermonkey-template/generate)。

## Development

```bash
yarn install
yarn run development
```

## Build

```bash
yarn run build
```
